Short Term Memory Test (Numbers)

Rules

Click Start to Start. A number appears on screen. Press no if you have not seen it, yes, if you have. There will be more and more numbers for you to keep track of and it will become difficult. Get 30 points to win, 0 lives and you are out. Have Fun!

Number Memory Test

Rules

Click Start and key in the number to Start. A number appears on screen, and it will last for 5 seconds. Memorise it. Afterwards there will be a textbox for you to fill in that number. If you get it right, you will gain 1 point and the number will increase to make it more difficult. Good Luck!
